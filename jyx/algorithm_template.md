# 常见算法模板

## 一、滑动窗口

最短子串模板

```c++
    for(int j=0 ; j<s.size() ; j++){
        窗口右端扩展，加进s[j],更新条件
        while(满足条件){
            和当前最优解进行比较并保存
            窗口左端移除s[i]，更新条件，i++
        }
    }
```

最长字串模板

```c++
    for (int j = 0; j < s.size(); j++) {
	    窗口右端扩展，加进s[j], 更新条件
	    while(不满足条件) {
		    窗口左端移除s[i]，更新条件，然后i++
	    }
	    此时重新满足条件，和最优比较并记录
    }
```

## 二、二分查找

查找从左往右最后一个满足条件的值

```c++
    while(l<r){
        int mid = (l+r+1)>>1;
        if(check(mid))
            l = mid;
        else
            r = mid-1;
    }
    return l;
```

查找从左往右第一个不满足条件的值

```c++
    while(l<r){
        int mid = (l+r)>>1;
        if(check(mid))
            r = mid;
        else
            l = mid+1;
    }
    return l;
```
